Αυτό το repository περιέχει την ερευνητική μας εργασία για το μάθημα "Τεχνικές Συμπίεσης Βίντεο". Η παρούσα εργασία διερευνά τις επιδόσεις των μεθόδων Μαθημένης Συμπίεσης Εικόνας (Learned Image Compression - LIC) σε ένα πλαίσιο σχετικό με την Κωδικοποίηση Βίντεο για Μηχανική Όραση (Video Coding for Machines - VCM). Στο VCM, ο στόχος της συμπίεσης μετατοπίζεται από την απλή οπτική ποιότητα για το ανθρώπινο μάτι (Rate-Distortion) στην ακρίβεια των εργασιών μηχανικής όρασης (Rate-Accuracy). Η εργασία έγινε ομαδικά στο πλαίσιο του μαθήματος.
